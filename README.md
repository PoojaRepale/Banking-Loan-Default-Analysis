# Bank Loan Default Analysis

## Overview
This project analyzes bank loan data to understand customer risk and predict loan defaults.

## Technologies Used
- Python
- Pandas
- NumPy
- Seaborn
- Matplotlib
- MySQL
- Power BI

## Project Structure
- data/
- notebooks/
- powerbi/
- sql/


## How to Run
1. Install requirements
2. Run notebooks
3. Load data in MySQL
4. Explore Power BI dashboard
